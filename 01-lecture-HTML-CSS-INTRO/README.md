# Instructions


## 1. Install [VS Code](https://code.visualstudio.com/). (We will work with the editor later)
## 2. Install [Git](https://git-scm.com/) and get familiar with the commands in [cli-commands.md](./cli-commands.md)
- Mac uses the internal terminal. Found by searching "terminal" in the search bar 
- PC uses Git Bash, the CLI that followed with the Git installation
## 3. Get familiar with VS code
- Workspace
- Views/tabs
- Search
- Extensions
- short commands in [vs-code-commands.md](./vs-code-commands.md)
## 4. HTML & CSS Intro
- Study the code html-intro
- Study the code css-intro
- Do the exercises in [profile-page-exercises.md](./profile-page-exercises.md)
## 5. Git & Github intro (If we have time)
- Create Github account: https://github.com/
- Configurations, to connect local git with github account
    - git config --global user.name "Your Name"
    - git config --global user.email "your_email@example.com"
    - git config --list 
- Simple versioncontroll through VS code
