# Visual Studio Code keyboard commands:

**Note!** `Cmd` is for MAC users. `Ctrl` is the equivalent for PC users (in place of `Cmd`).

---

### ‚å®Ô∏è Productivity & Editing

| Shortcut (MAC / PC Equivalent) | Description |
| :--- | :--- |
| `alt + shift + down` | Copies/pastes code below the current line |
| `cmd + d` / `Ctrl + d` | Multi-selects identical code (marks the next occurrence) |
| `alt + up/down` | Moves the selected line of code up/down |
| `alt + shift + mouse pointer` | Multi-cursor (vertical block selection) |
| `alt + mouse pointer` | Another type of multi-cursor (place cursor at any location) |

| Shortcut (MAC / PC Equivalent) | Description |
| :--- | :--- |
| `cmd + s` / `Ctrl + s` | Saves changes |
| `cmd + shift + 7` / `Ctrl + shift + 7` | Toggles block comment (`//` or `/* */`) |
| `cmd + f` / `Ctrl + f` | Searches within the specific file |
| `cmd + z` / `Ctrl + z` | Undoes changes |
| `cmd + shift + z` / `Ctrl + shift + z` | Redoes the undone changes |
